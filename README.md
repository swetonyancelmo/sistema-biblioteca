# Sistema de Biblioteca

Trabalho acadêmico de frontend para um sistema de gerenciamento de biblioteca. Desenvolvido com HTML, CSS e JavaScript puro, sem frameworks.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Estrutura

```
sistema-biblioteca/
├── index.html
├── css/
│   └── style.css
└── js/
    └── main.js
```
